Interface for the component that retrieves feature flag data.

The default implementation is \LaunchDarkly\Integrations\Guzzle::featureRequester(), which requests flags inefficiently via HTTP on demand. For other implementations, including database integrations, see the LaunchDarkly\Integrations namespace.
